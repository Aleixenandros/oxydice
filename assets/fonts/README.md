# Fuentes empotradas

RustNotes incrusta estas fuentes en el binario (`include_bytes!`) para
cumplir la guía de estilo §3.

| Fichero | Familia | Uso |
|---|---|---|
| `Inter-Regular.ttf` | Inter | Interfaz (texto, botones, etiquetas) |
| `Inter-SemiBold.ttf` | Inter SemiBold | Títulos / encabezados |
| `JetBrainsMono-Regular.ttf` | JetBrains Mono | Editor y monoespaciado |
